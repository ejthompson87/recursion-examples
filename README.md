Recursion Examples 
==================

###Testing - Jasmine###

Need to install Jasmine (detailed steps here - https://www.npmjs.com/package/jasmine)

###Installation:###

Local installation: 

    $ npm install --save-dev jasmine
 
Global installation:

    $ npm install -g jasmine

###Intializing###

To initialize a project for Jasmine

    $ jasmine init

###Usage###

To run your test suite

    $ jasmine
